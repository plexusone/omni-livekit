## [v0.1.0] - 2026-07-05

### Highlights

- Initial release of omni-livekit - LiveKit voice gateway for WebRTC-based voice AI applications
- OmniMeet provider implementation for LiveKit meeting abstraction
- OmniVoice gateway for voice AI pipeline integration
- Static image avatar support for voice agents with 16:9 canvas
